#Fear Lister in Angular JS with Angular ui-router
<h3>Author:</h3>
Jocelyn Alsdorf

<h3>Description:</h3>
This app allows a user to enter a fear, see all fears, and add more people who also have that fear. It uses Angular's ui-router to display partials.

<h2>Dependencies:</h2>

Angular JS v1.4.6
Angular ui-router
Bootstrap
